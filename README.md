# streaming_app

A new Flutter project.

## Getting Started

This Live Streaming app created in flutter.

Features:-
Register and Login with existing account.
Follow & Unfollow.
Group chat.
Personal chat.
Live Streaming.
Group Video Call.
Community Creation

Technology Used:-

Flutter
Dart
Getx
Agora
API

Packages:-

Dio
Google Fonts
Carosal Generator
